在没有给 flex 容器的项目设置属性的情况下，项目会使用默认样式
​
而与宽度相关的是下面几个：

  • flex-grow：项目的放大比例，默认是 0，即存在剩余空间，也不放大
  • flex-shrink：项目的缩小比例，默认为 1，即空间不足，项目将缩小
  • flex-basis：定义项目在主轴方向上的初始大小，默认值是 auto，即项目的本来大小

所有造成上面样式的原因就是 label 区域会进行缩小，空间被 value 区域占据

我应该怎样让 label 的区域不被侵占呢？首先可以设置 flex-shrink，当 flex-shrink 属性值为 0，而其他项目该属性为 1 时，如果空间不足时，属性值为 0 的项目不会缩小
